FEATURES

Swipe or Arrow keys

Fetch random waifu images instantly. Swipe on mobile, Press Arrow keys on desktop.

30+ SFW Categories

Waifu, neko, shinobu, megumin and many more. Browse by type or go random.

NSFW Mode

Age-verified NSFW content available. Toggled safely behind a verification gate.

Favorites Collection

Save your picks and download images directly to your device.

Auto-Slide Mode

Let waifus cycle automatically. Speed options: 2s, 3s, 5s, or 10s.

History Navigation

Go back and forward through your viewing history. Nothing gets lost.

Community Chat

Talk with anime fans in the built-in Talks community section.

Dark / Light Theme

Switch modes any time. Both are clean and easy on the eyes.

Native Apps

Available as native apps on Android, Windows, and Linux for a seamless experience.

HOW TO INSTALL

ANDROID APK
  1. Download the APK above
  2. Go to Settings → Security
  3. Enable Install Unknown Sources
  4. Open the APK and tap Install
  5. Launch and enjoy
WINDOWS
  1. Download the installer when available
  2. Run the .exe setup file
  3. Follow the installation wizard
  4. Launch SwipeWaifu from your desktop
LINUX

ABOUT PROJECT

What is SwipeWaifu?

SwipeWaifu is an open-source anime waifu discovery app built for anime enthusiasts. Swipe through 30+ SFW categories, save your favorites, use keyboard shortcuts on desktop, and chat with the Anime Fanbase community — all in one place. Available on Android, Web, and soon on Windows & Linux.

The project aims to be fast, fun, and free — celebrating anime culture through clean design and a constantly growing image library. Open source and community-driven, contributions are always welcome.
