HOW TO INSTALL

ANDROID APK
  1. Download the APK above
  2. Go to Settings → Security
  3. Enable Install Unknown Sources
  4. Open the APK and tap Install
  5. Launch and enjoy
WINDOWS
  1. Download the installer when available
  2. Run the .exe setup file
  3. Follow the installation wizard
  4. Launch SwipeWaifu from your desktop
LINUX
  1. Download the package when available
  2. Install via your package manager or .AppImage
  3. Make it executable if needed
  4. Launch and enjoy SwipeWaifu
